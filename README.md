# SCT_CS_03
Password Strength Checker (Tkinter GUI)

A Python desktop application that checks the strength of a password in real-time using **Tkinter**.  
It evaluates multiple criteria and provides instant feedback with a progress bar, labels, and color-coded indicators.

---

Features
-  Real-time password strength evaluation  
- Criteria checks:
  - Minimum 8 characters  
  - Contains uppercase letters  
  - Contains lowercase letters  
  - Contains numbers  
  - Contains special characters  
-  Visual feedback:
  - Progress bar showing strength percentage  
  - Strength label (`Very Weak`, `Weak`, `Medium`, `Strong`, `Excellent`)  
  - Color-coded criteria (green or red )  
-  Show/Hide password toggle
